Introduction to html and elements

HTML tags:-

HyperText Markup Language (HTML) is the standard markup language for documents designed to be displayed in a web browser. It defines the content and structure of web content. It is often assisted by technologies such as Cascading Style Sheets (CSS) and scripting languages such as JavaScript.

(1)b tag

The HTML tag is used to make text bold. It doesn't carry any semantic meaning on its own; it's just a presentational element. When you enclose text within and tags, the text becomes visually bold on the webpage.

eg.Hyper text markup language

(2)i tag

The tag in HTML is used to display the content in italic style. This tag is generally used to display the technical term, phrase, the important word in a different language. The tag is a container tag that contains the opening tag, content & closing tag.

eg.Hyper text markup language

(3)u tag

The HTML tag stands for underline which helps to underline the text enclosed within the tag. Underlining the text is mainly done to indicate the unarticulated or styled differently from normal text.


eg.Hyper text markup language
